## 🎯 使用方式

### 方式一：右键菜单（最直观）

#### 在编辑器中右键点击

**场景1：选中代码后右键**
```
├── 小尾巴: 解释代码          ← 解释选中的代码
```

### 条件显示逻辑

| 命令 | 显示条件 |
|------|---------|
| 解释代码 | `editorHasSelection` (有选中文本) |
| 检查命名 | `editorHasSelection && editorLangId =~ /代码语言/` |
| 生成代码 | `editorTextFocus` (编辑器有焦点) |
| 优化 SQL | `editorHasSelection && editorLangId == sql` |
| 查看提交历史 | `editorTextFocus` (编辑器有焦点) |

## 🔧 自定义配置

### 修改右键菜单顺序

编辑 `package.json` 中的 `menus.editor/context` 部分：

```json
{
  "command": "xiaoweiba.explainCode",
  "group": "xiaoweiba@1"  // 数字越小，位置越靠前
}
```

---

### 隐藏不需要的命令

在 `package.json` 的 `menus.commandPalette` 中添加 `"when": "false"`：

```json
{
  "command": "xiaoweiba.repairMemory",
  "when": "false"  // 不在命令面板中显示
}
```

---

### 添加新的快捷键（可选）

虽然目标是摆脱快捷键，但如果某些高频操作您仍希望保留快捷键，可以在 `package.json` 的 `keybindings` 中添加：

```json
{
  "command": "xiaoweiba.explainCode",
  "key": "ctrl+shift+e",
  "when": "editorHasSelection"
}
```
